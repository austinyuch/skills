# 生命權平等協會 NPO 營銷策略指南

**版本**：v1.0  
**適用對象**：生命權平等協會及類似議題NPO  
**核心目標**：推廣生命權平等觀點，推動司法改革，建立公眾支持

---

## 一、品牌定位與核心信息

### 1.1 議題品牌架構

**什麼是生命權平等？**
```
電梯演講版（30秒）：
「生命權平等是指每個生命都應該受到司法制度的平等保護。當犯罪發生時，
我們的制度不應該只關注加害者，更要保障被害人及其家屬的權益，
讓他們能參與司法程序、獲得賠償與心理支持。」

簡短版（2分鐘）：
[生命權平等的3個核心要素]
1. 訴訟參與權：被害人應有權利參與司法程序，表達意見
2. 資訊透明權：案件進度、判決理由應清楚告知被害人
3. 援助支持權：法律協助、心理諮商、經濟賠償應獲得保障

完整版：
[參考組織使命宣言與政策白皮書]
```

**核心價值主張（Value Proposition）**：
```
我們不是反對司法保護加害者，
而是要求司法**同時**保護被害者。
正義不應該是零和遊戲。
```

### 1.2 視覺識別系統建議

**品牌個性**：
- 專業但不冷漠
- 堅定但不對立
- 溫暖但不煽情
- 希望但不輕率

**視覺元素**：
| 元素 | 建議 | 原因 |
|------|------|------|
| **主色** | 深藍色（#1a365d） | 信任、專業、司法 |
| **輔色** | 暖橙色（#ed8936） | 希望、行動、人性 |
| **強調色** | 白色 | 純粹、平等、清晰 |
| **字體** | 思源黑體/Noto Sans | 易讀、中性、專業 |
| **圖像風格** | 真實人物照+簡潔圖表 | 人性化+專業性 |

**Logo概念建議**：
- 天平（正義）+ 雙手（保護）+ 光芒（希望）
- 避免過度複雜，確保在小尺寸社群頭像中可識別

## 二、受眾分群策略（Audience Segmentation）

### 2.1 五大受眾群體

| 群體 | 特徵 | 關注點 | 行動門檻 | 溝通策略 |
|------|------|--------|----------|----------|
| **A. 司法專業人士** | 法官、檢察官、律師、學者 | 制度可行性、法律條文、實務操作 | 高（需改變工作慣例） | 專業論述、研究數據、試點方案 |
| **B. 潛在受害者** | 曾受害或擔心受害的民眾 | 自身權益保障、如何尋求協助 | 中（關注但不一定行動） | 實用資訊、成功案例、支持網絡 |
| **C. 社會關注者** | 關注社會議題的一般大眾 | 正義、公平、社會進步 | 低（分享、連署） | 情感故事、簡易懶解、行動呼籲 |
| **D. 政策影響者** | 立委、官員、政黨、媒體 | 選票、輿論、專業評價 | 高（需政治判斷） | 數據報告、民意展現、背書聯盟 |
| **E. 潛在支持者** | 可能捐款或當志工的人 | 組織可信度、影響力、透明度 | 中（金錢或時間投入） | 成果報告、財務透明、參與機會 |

## 三、內容營銷矩陣（Content Marketing Matrix）

### 3.1 內容類型策略

建立「4E內容模型」：

| 類型 | 目的 | 比例 | 範例 |
|------|------|------|------|
| **Educate 教育** | 建立議題認知 | 30% | 司法制度知識普及、被害人權益說明 |
| **Engage 互動** | 建立情感連結 | 40% | 家屬故事、活動現場、讀者問答 |
| **Empower 賦權** | 促進行動參與 | 20% | 連署活動、志工招募、捐款呼籲 |
| **Evidence 證據** | 建立專業可信 | 10% | 研究報告、數據分析、專家觀點 |

### 3.2 內容製作流程

```
週一：編輯會議（選定本週主題）
  ↓
週二-三：內容製作（social-post-generator生成草稿）
  ↓
週四：內容審核（事實查核+法律檢查）
  ↓
週五-日：發布與互動（排程發布+即時回應）
```

### 3.3 內容再利用策略

**一次製作，多次使用**：

| 原始內容 | 再利用形式 | 適用平台 |
|---------|-----------|----------|
| 深度報告 | 1張資訊圖表 + 3篇短文 + 1支短片 | IG圖文 + FB貼文 + YT Shorts |
| 記者會 | 5張金句圖卡 + 1篇逐字稿 + 3段短片 | IG限動 + 網站 + Reels |
| 家屬訪談 | 1支3分鐘影片 + 5張照片 + 3則引述 | YT + IG + FB |
| 修法進展 | 1張時間線圖 + 1篇說明文 + 行動呼籲 | FB + 網站 |

## 四、多渠道傳播策略

### 4.1 官方網站（核心基地）

**必要功能**：
- 議題介紹（生命權平等是什麼）
- 最新新聞（victim-rights-news-tracker整合）
- 資源下載（被害人權益手冊PDF）
- 行動參與（連署、捐款、志工）
- 關於我們（組織介紹、財報、聯絡）

**SEO優化關鍵字**：
```
主要：被害人權益、司法改革、生命權平等
次要：被害人訴訟參與、刑事被害人保護法、家屬權益
長尾：車禍被害人怎麼辦、判決後家屬能怎麼做、被害人可訴請賠償嗎
```

### 4.2 社群媒體矩陣

| 平台 | 角色定位 | 內容頻率 | 主要功能 |
|------|---------|---------|----------|
| **Facebook** | 主要發布平台 | 每日1-2則 | 深度內容、活動宣傳、社群討論 |
| **Instagram** | 視覺品牌建立 | 每日1則 | 家屬故事、活動現場、限時動態 |
| **Threads** | 輕鬆互動 | 每日2-3則 | 即時回應、話題跟進、對話討論 |
| **YouTube** | 深度內容 | 每月2-4支 | 訪談紀錄片、議題懶說影片 |
| **Line官方** | 核心支持者 | 每週2-3則 | 電子報、活動通知、獨家內容 |

**跨平台內容調整範例**：

```
原始內容：修法記者會新聞稿（500字）

→ Facebook（完整版）：
   標題 + 全文 + 活動照片 + 連署連結
   
→ Instagram（視覺版）：
   單張圖表（關鍵3要點）+ 簡短說明 + 限動連續貼文
   
→ Threads（對話版）：
   「大家聽說修法的新進展了嗎？🚨
   簡單來說就是... [3點懶說]
   你覺得這次能過嗎？👇」
   
→ YouTube（深度版）：
   3分鐘影片：理事長訪談 + 懶說動畫 + 呼籲行動
```

### 4.3 傳統媒體關係

**媒體關係建立**：
- 建立記者聯絡清單（司法線、社會線、政治線）
- 定期提供新聞素材包（數據、故事、專家）
- 主動提供獨家訪談機會
- 快速回應媒體詢問（24小時內）

**新聞稿範本結構**：
```
標題：[行動動詞] + [具體事實] + [影響力]
範例：「生命權平等協會公布：8成家屬認為現行制度忽視被害人權益」

第一段：
- 誰（什麼事）
- 何時
- 誰說的
- 為什麼重要

第二段：
- 背景脈絡
- 具體數據/引用

第三段：
- 專家/家屬引述

第四段：
- 行動呼籲
- 聯絡資訊
```

### 4.4 Podcast/直播策略

**Podcast節目建議**：
```
節目名稱：《正義的聲音》或《生命權平等說》
頻率：每週1集，每集30-45分鐘
形式：訪談家屬、專家、律師、立委

集數規劃：
EP1：什麼是生命權平等？（理事長訪談）
EP2：修法之路（歷程回顧）
EP3：家屬現身說法（小燈泡案母親）
EP4：法界觀點（律師/法官訪談）
EP5：國際比較（其他國家怎麼做）
...
```

**直播策略**：
- 修法現場直播（立法院審查時）
- 每月1次「問與答」直播（回應網友問題）
- 家屬分享直播（特殊紀念日）

## 五、建立合作夥伴網絡

### 5.1 策略聯盟類型

| 類型 | 對象 | 合作形式 | 互利點 |
|------|------|---------|--------|
| **議題聯盟** | 其他司法改革NPO | 聯合聲明、共同活動 | 資源整合、擴大影響 |
| **專業背書** | 律師公會、法學教授 | 研究合作、顧問諮詢 | 專業可信度 |
| **民意代表** | 支持修法立委 | 法案推動、質詢合作 | 政策影響力 |
| **媒體合作** | 網路媒體、Podcast | 專欄合作、專訪安排 | 曝光管道 |

### 5.2 合作夥伴關係維護

**定期溝通機制**：
- 每季1次合作夥伴聚會
- 每月電子報發送（分享工作進展）
- 重要時刻即時通知（法案進展、新聞曝光）

**給予價值，不只索取**：
- 提供獨家內容給合作夥伴使用
- 在公開場合感謝背書者
- 協助夥伴達成其目標

## 六、影響者/意見領袖（KOL）策略

### 6.1 目標KOL類型

| 類型 | 範例 | 合作方式 | 預期效益 |
|------|------|---------|----------|
| **司法專家KOL** | 知名律師、法普YouTuber | 專訪、共同直播 | 專業背書、法界人士觸及 |
| **社會議題KOL** | 社會評論者、公民記者 | 議題合作、活動出席 | 議題擴散、社會關注 |
| **人文關懷KOL** | 心理師、作家、主持人 | 家屬故事合作 | 情感共鳴、中間選民 |
| **政治評論KOL** | 政治評論節目、專欄作家 | 政策分析合作 | 政治圈影響、政策推動 |

### 6.2 KOL合作原則

**選擇標準**：
- ✅ 價值觀契合（支持被害人權益）
- ✅ 受眾與目標群體重疊
- ✅ 過往內容品質良好
- ✅ 願意深入了解議題

**避免合作**：
- ❌ 立場極端或爭議性人物
- ❌ 只追求流量不願深入了解
- ❌ 形象與組織調性不符

**合作模式**：
1. **內容合作**：邀請KOL訪談家屬或參與活動
2. **聯名倡議**：KOL簽名支持修法聲明
3. **知識贊助**：KOL提供專業知識內容
4. **資源贊助**：KOL捐款或號召粉絲支持

```

            ↓
    邀請家屬接受訪談（經授權）
```

## 八、數據驅動的優化

### 8.1 關鍵指標（KPIs）

**認知度指標**：
- 網站每月訪問人次
- 社群媒體觸及人數（Reach）
- 媒體報導次數
- 搜尋引擎排名（關鍵字：被害人權益）

**支持度指標**：
- 連署人數
- 電子報訂閱人數
- 社群互動率（讚、留言、分享）
- 活動參與人數

**行動度指標**：
- 捐款人數與金額
- 志工人數
- 政策改變進展（法案通過進度）

**忠誠度指標**：
- 重複捐款率
- 志工持續服務時間
- 電子報開信率

### 8.2 每月分析檢討

**分析報告範本**：
```
【月份】營銷成效報告

1. 內容表現TOP3
   - 最高互動貼文：[標題]（互動數）
   - 最高分享貼文：[標題]（分享數）
   - 最高觸及貼文：[標題]（觸及數）

2. 受眾成長
   - 新增追蹤：+X人（成長率%）
   - 電子報訂閱：+X人
   - 網站流量：X人次（+/-%）

3. 重要成果
   - [具體成果1]
   - [具體成果2]

4. 下月調整
   - 多做：[根據數據，什麼內容效果好]
   - 少做：[什麼內容效果差]
   - 測試：[新嘗試]
```

### 8.3 A/B測試項目

**可測試元素**：
- 貼文標題（情感 vs 理性）
- 發布時間（早上 vs 晚上）
- CTA按鈕文字（「立即連署」vs 「支持修法」）
- 圖片類型（人物照片 vs 資訊圖表）
- Hashtag數量（多 vs 少）

## 九、資源配置與優先序

### 9.1 不同規模的起步建議

**草創期（0-6個月）**：
- 人力：1-2人（執行長+內容/社群）
- 重點：建立基礎平台、產出穩定內容、累積首批支持者
- 工具：social-post-generator（每日發文）、簡單網站、FB/IG帳號

**成長期（6-18個月）**：
- 人力：3-5人（加募款、活動、媒體關係）
- 重點：擴大影響力、建立媒體關係、舉辦實體活動
- 工具：victim-rights-news-tracker（新聞監控）、CRM系統

**成熟期（18個月+）**：
- 人力：5-10人
- 重點：專業分工、政策推動、跨組織合作
- 工具：自動化行銷工具、數據分析平台

### 9.2 預算分配建議（參考）

**小型NPO（年預算<100萬）**：
- 內容製作：30%（影片、設計）
- 數位廣告：25%（FB廣告、關鍵字）
- 活動執行：25%（場地、材料）
- 工具/平台：10%
- 其他：10%

**中型NPO（年預算100-500萬）**：
- 內容製作：25%
- 數位廣告：20%
- 活動執行：25%
