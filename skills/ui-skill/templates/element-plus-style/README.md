# Element Plus風格模板

## 風格特點

- **主色调**：#409EFF（天蓝色）
- **設計風格**：清新、友好、易用
- **視觉效果**：柔和、舒適、亲和

## 適用場景

- 中後台系統
- 内部工具和管理系統
- 需要友好介面的應用
- 中小型團隊使用的系統

## 模板檔案

- `template.html` - 完整的Element Plus風格B端供應鏈採購订單管理頁面模板

## 模板說明

此模板完整實作了採購订單管理的所有功能,包括:

### 核心功能
- 统計卡片展示
- 多條件篩選搜尋
- 订單清單格展示
- 完整的CRUD操作(新增、檢視、編輯、刪除)
- 批量操作(批量审批、批量刪除)
- 資料匯入匯出
- 订單审批流程
- 表格單驗證

### 技術特點
- 單檔案HTML,包含完整的CSS和JavaScript
- 使用模擬資料,無需後端支援
- 所有互動功能完全可用
- 回應式設計,支援多種屏幕尺寸
- 程式碼結構清晰,註解完善

### 使用方式
直接在浏覽器中打开`template.html`即可檢視和使用。

### 與其他模板的关系
- **HTML結構和JavaScript功能**與at-ui-style和ant-design-style模板**完全一致**
- **仅CSS樣式不同**,采用Element Plus設計系統的視觉規範
- 可以無缝切换到其他風格的模板
