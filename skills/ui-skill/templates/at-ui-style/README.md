# AT-UI風格模板

## 風格特點

- **主色调**：#E94609（橙红色）
- **設計風格**：現代、專業、動感
- **視觉效果**：渐變按钮、卡片投影、丰富的動画效果

## 組件特色

### 按钮
- 渐變背景（#E94609 → #D73A00）
- 悬停光泽動画
- 微妙的阴影效果

### 卡片
- 左侧色条装饰
- 悬停提升效果
- 渐變阴影

### 狀態標籤
- 渐變背景
- 圆点圖標 + 脉冲動画
- 多種狀態颜色

### 表格格
- 悬停行高亮
- 微小提升效果
- 平滑过渡動画

### 模態框
- 毛玻璃背景
- 滑入動画
- 圆角設計

## 適用場景

- 通用B端管理系統
- 需要現代化視觉效果的項目
- 強调品牌色彩的系統
- 需要丰富互動反馈的應用

## 參考示例

- 採購订單管理系統
- 供應商管理系統
- 庫存管理系統

## 技術特點

- 纯HTML + CSS + JavaScript
- 無需建置工具
- 直接在浏覽器中執行
- 完整的CRUD互動
- 回應式設計

## 使用方法

在程式碼產生階段，選擇此模板，系統会基於template.html產生您需要的頁面。
