# Hashtag 策略與分類建議 (Hashtag Strategy)

本文檔提供社群發文的 Hashtag 策略，包含分類建議、使用時機與最佳實踐。

---

## 一、Hashtag 分類架構

### 1.1 核心標籤（必用）

**組織識別標籤**
- `#生命權平等協會` - 組織主要標籤
- `#生命權平等` - 組織願景標籤

**使命標籤**
- `#被害人權益` - 核心使命
- `#家屬權益` - 關注對象
- `#司法正義` - 目標願景

### 1.2 主題標籤（依內容選用）

| 主題類別 | 標籤建議 | 使用時機 |
|---------|---------|----------|
| **司法改革** | `#司法改革` `#制度正義` `#修法倡議` | 制度變革相關 |
| **案件追蹤** | `#[案件名稱]` `#案件追蹤` | 特定案件報導 |
| **家屬聲音** | `#家屬的聲音` `#陪伴` `#正義不缺席` | 家屬故事為主 |
| **修法行動** | `#支持修法` `#一起行動` `#改變從你開始` | 行動呼籲 |
| **人權保障** | `#人權` `#公民權利` `#弱勢保障` | 人權角度論述 |
| **社會關注** | `#社會議題` `#關注司法` `#台灣社會` | 擴大社會關注 |

### 1.3 功能標籤（依目的選用）

**行動導向**
- `#立即分享` `#一起參與` `#連署支持`
- `#打電話` `#行動改變` `#現在就行動`

**情感導向**
- `#溫暖` `#陪伴` `#不孤單`
- `#記得# #懷念# #愛與正義`

**資訊導向**
- `#新聞快訊` `#修法進展` `#判決結果`
- `#重要消息` `#資訊分享` `#關注更新`

## 二、平台特定策略

### 2.1 Facebook

**特性**：可放較多標籤，但過多會顯得雜亂

**建議數量**：8-12個

**組合策略**：
```
#生命權平等協會 #被害人權益 #司法改革 
#[案件名稱] #修法倡議 #制度正義 
#一起行動 #改變從你開始 #台灣社會
```

**放置位置**：貼文結尾或第一則留言

### 2.2 Instagram

**特性**：標籤影響觸及率，但前2行最重要

**建議數量**：5-8個

**組合策略**：
```
貼文內容...（前2行要吸睛）

.
.
.
#生命權平等 #被害人權益 #司法改革 
#[案件名稱] #修法倡議 #台灣
```

**放置技巧**：
- 使用換行符號（.
.
.
）隔離標籤
- 或放在第一則留言

### 2.3 Threads

**特性**：輕鬆氛圍，標籤不宜過多

**建議數量**：3-5個

**組合策略**：
```
內容...你覺得呢？

#生命權平等 #司法改革 #[案件名稱]
```

**放置位置**：直接融入內文結尾

## 四、標籤使用禁忌

### ❌ 不應該做的事

**過度使用**
- 單一貼文超過15個標籤（看起來像垃圾訊息）
- 使用與內容無關的熱門標籤（騙取曝光）
- 在留言區連續發布大量標籤

**錯誤使用**
- 拼寫錯誤（如`#司法改格`）
- 使用負面或爭議性標籤
- 使用可能誤導的標籤

**道德問題**
- 使用悲劇事件的娛樂化標籤
- 消費受害者標籤
- 未經授權使用私人標籤

| 指標 | 評估方式 | 改善方向 |

| 新追蹤來源 | 檢視新追蹤者從哪個標籤來 | 加強有效標籤 |

## 六、動態標籤建議

### 6.1 依日期/事件的變化標籤

**固定節日**：
- 司改紀念日：`#司法改革紀念日`
- 人權日：`#世界人權日` `#人權`

**案件特定**：
- 小燈泡案：`#小燈泡案` `#王婉諭`
- 特定日期：`#[案件名稱]周年`

### 6.2 趨勢標籤跟隨

**原則**：
- 只在趨勢標籤與組織使命相關時使用
- 不要為了跟風使用無關標籤
- 使用趨勢標籤時，內容要真的有貢獻

**範例**：
```
如果趨勢是 #MeToo
相關貼文：「#MeToo 運動凸顯被害人發聲的重要性...」
無關貼文：「支持修法 #MeToo」（不適當）
```
