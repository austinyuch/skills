# B端供應鏈業務术语表

本文件定義B端供應鏈管理中常用的專業术语，確保PRD文件中术语使用的准确性和一致性。

## 採購管理术语

| 术语 | 英文 | 定義 | 應用場景 |
|------|------|------|----------|
| 採購申请 | Purchase Requisition | 業務部门提出的物料採購需求單据 | 需求发起、预算管理 |
| 採購订單 | Purchase Order (PO) | 採購部门向供應商发出的正式採購凭证 | 物料採購、服务採購 |
| 詢价單 | Request for Quotation (RFQ) | 向多個供應商詢问价格的單据 | 价格比较、供應商選擇 |
| 报价單 | Quotation | 供應商针對詢价提供的价格方案 | 价格評估、商务谈判 |
| 採購合同 | Procurement Contract | 与供應商签订的正式採購协議 | 合同管理、风險控制 |
| 採購周期 | Procurement Cycle | 從提出採購需求到物料到货的时间 | 效率評估、计划制定 |
| 採購成本 | Procurement Cost | 採購物料的总支出，包括价格、運費等 | 成本控制、预算管理 |
| 供應商准入 | Supplier Onboarding | 供應商进入合格供應商庫的審核過程 | 供應商管理、品質控制 |
| 供應商評級 | Supplier Rating | 基於品質、交期、价格、服务等維度的供應商综合評分 | 供應商准入、績效考核 |
| 供應商績效 | Supplier Performance | 供應商在合作過程中的表现和指标 | 績效管理、優化調整 |
| 框架协議 | Framework Agreement | 与供應商签订的长期合作框架合同 | 戰略採購、批量优惠 |
| 比价採購 | Competitive Bidding | 透過多家供應商比价選擇最优方案 | 降本增效、透明採購 |

## 庫存管理术语

| 术语 | 英文 | 定義 | 應用場景 |
|------|------|------|----------|
| 庫存 | Inventory/Stock | 企业为生产经营目的而储备的物料 | 庫存管理、资产管理 |
| 庫存周转率 | Inventory Turnover | 一定时期内庫存物料的周转次数，反映庫存管理效率 | 庫存優化、成本控制 |
| 庫存周转天数 | Days Inventory Outstanding (DIO) | 庫存從入庫到出庫的平均天数 | 效率分析、资金占用 |
| 安全庫存 | Safety Stock | 为防止缺货而保持的最低庫存量 | 庫存计划、风險控制 |
| 最高庫存 | Maximum Stock Level | 庫存不應超过的最高数量 | 仓储管理、成本控制 |
| 最低庫存 | Minimum Stock Level | 觸發採購补货的庫存数量 | 补货管理、供應保障 |
| 庫存预警 | Stock Alert | 庫存达到预设阈值时的系統提醒 | 风險管理、及时补货 |
| 呆滞庫存 | Slow-Moving Stock | 长期未動用或周转缓慢的庫存 | 庫存優化、资产盘活 |
| 账实差异 | Stock Discrepancy | 账面庫存与實際庫存的差异 | 盘点管理、损耗控制 |
| ABC分类 | ABC Classification | 按價值重要性将物料分为A、B、C三类的管理方法 | 庫存分类、採購策略 |
| 先进先出 | First In First Out (FIFO) | 先入庫的物料先出庫的管理原則 | 庫存管理、品質控制 |
| 庫存盘点 | Stock Taking | 定期對庫存进行实物清点核對 | 准确性管理、损耗控制 |
| 收货入庫 | Goods Receipt | 物料到货后的入庫登记操作 | 入庫管理、庫存增加 |
| 发货出庫 | Goods Issue | 物料出庫发给使用部门的操作 | 出庫管理、庫存减少 |
| 庫存调拨 | Stock Transfer | 庫存在不同仓庫或地点之间的移動 | 庫存调配、資源優化 |
| 庫存冻结 | Stock Freeze | 将特定庫存設定为不可用狀態 | 品質管控、纠纷處理 |

## 物流管理术语

| 术语 | 英文 | 定義 | 應用場景 |
|------|------|------|----------|
| 物流 | Logistics | 物品從供應地到接收地的實體流動過程 | 運輸管理、配送管理 |
| 运單 | Waybill | 承运人出具的貨物運輸凭证 | 物流跟踪、運費結算 |
| 配送單 | Delivery Note | 配送貨物给客户的單据 | 配送管理、签收确认 |
| 签收單 | Receipt | 收货方确认收到貨物的凭证 | 交付确认、對账依据 |
| 運費 | Freight | 貨物運輸所需支付的费用 | 成本核算、费用結算 |
| 運輸时效 | Transit Time | 貨物從发出到到达的时间 | 服务水平、客户满意度 |
| 物流跟踪 | Shipment Tracking | 实时監控貨物運輸狀態和位置 | 可视化管理、异常预警 |
| 在途庫存 | In-Transit Inventory | 已发出但尚未到达的貨物 | 庫存管理、资金占用 |
| 物流异常 | Logistics Exception | 運輸過程中的延误、损坏等問題 | 异常處理、客户服务 |
| 到货检验 | Incoming Inspection | 對供應商交付物料的品質、数量、规格进行确认 | 入庫管理、品質控制 |
| 退貨物流 | Return Logistics | 不合格品或其他原因退回供應商的物流 | 退货管理、逆向物流 |
| 運輸方式 | Mode of Transport | 貨物運輸采用的方式（陆运、海运、空运等） | 物流規劃、成本優化 |

## 品質管理术语

| 术语 | 英文 | 定義 | 應用場景 |
|------|------|------|----------|
| 品質標準 | Quality Standard | 物料應达到的品質要求和規範 | 品質管控、供應商管理 |
| 质检報告 | Quality Inspection Report | 品質检验的结果記錄 | 品質追溯、問題分析 |
| 合格率 | Qualified Rate | 合格物料占总检验物料的比例 | 績效評估、品質改进 |
| 不合格品 | Non-Conforming Product | 未达到品質標準的物料 | 品質控制、退货處理 |
| 让步接收 | Concession Acceptance | 對轻微不合格品的有條件接收 | 灵活處理、成本平衡 |
| 品質整改 | Quality Improvement Action | 针對品質問題的改进措施 | 品質提升、供應商管理 |
| 品質追溯 | Quality Traceability | 追踪物料的品質来源和流向 | 問題定位、责任界定 |
| 抽检 | Sampling Inspection | 從批次中抽取样品进行检验 | 品質控制、效率平衡 |
| 全检 | 100% Inspection | 對批次中所有物料逐一检验 | 嚴格质控、高风險物料 |

## 术语使用規範

### 1. 一致性原則

- 同一术语在整個PRD文件中使用一致
- 优先使用中文术语，必要时注明英文
- 特殊行業术语需在"業務名词"章節解释

### 2. 准确性原則

- 使用供應鏈行業標準术语
- 避免自造术语或模糊表达
- 參考行業規範和最佳實務

### 3. 可理解性原則

- 首次出现的專業术语要解释
- 複雜术语提供使用場景说明
- 面向非專業人员时适當简化
