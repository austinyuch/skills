# Industry Reference Candidates

This file is a parking lot for industry-specific AI and security references. These sources are not part of the default audit workflow yet. Use them as official/primary leads when a future task explicitly asks to add an industry-specific lens.

NIST AI RMF, the AI RMF Playbook, and NIST AI 600-1 are no longer parked here because they are part of the default cross-sector AI baseline in `legal-regulatory-map.md` and `minimum-control-map.md`.

## Cross-Sector AI and Cybersecurity

| Area | Source URL | Notes |
|---|---|---|
| Cybersecurity risk management | `https://www.nist.gov/cyberframework` | NIST CSF 2.0; useful cross-sector security baseline |
| Critical infrastructure cybersecurity goals | `https://www.cisa.gov/cross-sector-cybersecurity-performance-goals` | CISA CPGs; useful for critical infrastructure minimum security outcomes |
| NIST SP 800-53 controls | `https://csrc.nist.gov/publications/detail/sp/800-53/rev-5/final` | Security and privacy controls for US federal/public-sector style control catalogs |
