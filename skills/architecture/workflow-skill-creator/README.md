# Workflow Skill Creator

專業的Workflow Skill產生器，幫助您快速建立高品質的AI工作流Skill。

## 簡介

Workflow Skill Creator基於Writer、PRD產生器、Kiro等成功案例的深度分析，掌握六階段工作流的核心設計模式。透過系統化的四階段流程，引導您從需求分析到最終交付，建立專業級的AI工作流Skill。

## 使用方式

这個Skill会被Claude自動调用，當您的请求匹配以下場景时：

- "我想建立一個workflow skill..."
- "帮我產生一個工作流提示词..."
- "需要設計一個AI工作流程..."
- "建立一個类似Writer/PRD/Kiro的skill..."

## 四階段工作流程

### 第一階段：需求分析与角色定位

- 透過系統化问卷收集基础信息
- 深度访谈瞭解行業特殊性和專業要求
- 產生完整的需求分析文件
- 明确AI助手的角色定位和核心價值主张

### 第二階段：六階段流程設計

- 設計符合需求的六個工作階段框架
- 为每個階段定義清晰的目标、輸入、輸出
- 設計專業的多維度評估体系
- 設計流程控制邏輯和階段推进机制

### 第三階段：Skill完整產生

- 基於通用模板进行系統化的變量替换
- 產生符合Agent Skill規範的SKILL.md主文件
- 產生六個階段的詳細實施指南（phases目錄）
- 產生必要的參考文件（reference目錄）

### 第四階段：预覽優化与交付

- 展示產生Skill的關鍵部分供预覽
- 进行六維度專業品質檢查
- 根据檢查结果进行迭代優化
- 最終交付完整、高品質的Workflow Skill

## 品質標準

產生的Skill经过六個維度的專業檢查：

1. **完整性** (20%) - 所有必需文件和章節齐全
2. **邏輯性** (25%) - 流程合理，階段衔接顺畅
3. **專業性** (25%) - 领域知识准确，术语專業
4. **可用性** (15%) - 格式正确，可直接使用
5. **创新性** (10%) - 有独特設計亮点
6. **規範性** (5%) - 符合Agent Skill規範

综合得分≥8.0即可交付，≥9.0为卓越水平。

## 常见問題

### Q: 產生一個Skill需要多长时间？

A: 通常需要15-30分钟，取决于：

- 需求的複雜度
- 沟通的效率
- 定制的程度

### Q: 可以產生非六階段的Skill吗？

A: 本工具专门用于產生六階段工作流Skill。如果您的需求不适合六階段結構，可能需要其他方式。

### Q: 產生的Skill可以修改吗？

A: 當然！產生的Skill是完全可編輯的。您可以：

- 直接編輯SKILL.md和其他文件
- 調整階段設計
- 修改評估維度
- 優化交互方式

### Q: 如何让產生的Skill更專業？

A: 關鍵在于第一階段的需求分析：

- 提供詳細的领域信息
- 说明行業特殊性
- 提供實際案例
- 明确品質標準

### Q: 產生后如何測試？

A: 建議的測試方法：

1. 詢问Claude："What skills are available?"
2. 提出匹配的请求，看是否觸發
3. 完整走一遍工作流
4. 檢查各階段文件產生

### Q: 如何与团队分享？

A: 两种方式：

1. **項目Skill**：提交到 `.opencode/skills/` 目錄
2. **插件分发**：建立OpenCode插件
