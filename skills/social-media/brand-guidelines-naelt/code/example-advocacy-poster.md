# 倡議文宣設計指南

## 設計原則

### 1. 視覺衝擊力
- **主視覺**: 使用強烈的視覺元素（照片、插圖、圖表）
- **色彩對比**: 紅色主色 + 白色背景 = 強烈對比
- **留白**: 適當留白讓視覺焦點更突出

### 2. 訊息層次
- **標題**: 簡短有力（≤10 字）
- **副標題**: 補充說明（≤20 字）
- **內文**: 條列式重點（3-5 點）
- **CTA**: 明確的行動指示

### 3. 情感連結
- **真實故事**: 使用真實案例和數據
- **人物照片**: 展現真實情感
- **溫暖色調**: 橙色強調希望與支持

## 海報尺寸規格

### 印刷品
- **A4**: 210mm × 297mm (300 DPI)
- **A3**: 297mm × 420mm (300 DPI)
- **A2**: 420mm × 594mm (300 DPI)
- **大型海報**: 841mm × 1189mm (150 DPI)

### 數位版
- **社群媒體**: 1080px × 1080px (Instagram)
- **橫幅**: 1920px × 1080px (網站 Banner)
- **直式**: 1080px × 1920px (限時動態)

## 設計範本

### 標題區
```
[主視覺圖片 - 佔 50% 高度]

[標題 - 6xl, bold, primary-600]
為生命權平等而戰

[副標題 - 2xl, normal, secondary-600]
推動九大修法訴求，保障被害人權益
```

### 內容區
```
[區塊標題 - 3xl, semibold, primary-600]

[條列內容 - lg, normal, secondary-800]

✓ 保障被害人權益
✓ 司法透明化

```

### CTA 區
```
[行動呼籲 - 2xl, bold, accent-500]
立即連署支持

[QR Code + 網址]
www.naelt.org/petition
```

## 色彩運用

### 主色調配置
- **背景**: 白色 (#FFFFFF) 或淺灰 (secondary-50)
- **標題**: 紅色 (primary-600) 或深灰 (secondary-800)
- **強調**: 橙色 (accent-500)
- **內文**: 深灰 (secondary-800)

### 情感色彩
- **緊急/重要**: 紅色 (primary-500)
- **希望/支持**: 橙色 (accent-500)
- **專業/穩定**: 灰色 (secondary-600)

## 圖片處理

### 攝影風格
- **真實**: 避免過度修圖
- **情感**: 捕捉真實情感
- **構圖**: 人物為主，環境為輔

### 色調調整
- 飽和度: -10%
- 對比度: +5%
- 色溫: 略微偏暖

### 疊加效果
- 漸層遮罩: 從透明到 primary-900 (opacity 60%)
- 確保文字可讀性

## 文案撰寫

### 標題原則
- **簡短**: ≤10 字
- **有力**: 使用動詞
- **情感**: 觸動人心
- **範例**: 
  - "為生命權平等而戰"
  - "正義不能等待"
  - "守護每一個生命"

### 內文原則
- **條列式**: 3-5 個重點
- **數據支持**: 使用具體數字
- **行動導向**: 告訴讀者可以做什麼
- **範例**:
  - "每年超過 1,000 個家庭受害"
  - "加入我們，推動司法改革"
  - "您的連署，就是改變的力量"

### CTA 原則
- **明確**: 清楚告訴讀者下一步
- **簡單**: 降低行動門檻
- **緊迫**: 創造行動動機
- **範例**:
  - "立即連署"
  - "加入志工"
  - "分享給朋友"

## 無障礙設計

### 色彩對比
- 標題與背景: 對比度 ≥ 4.5:1
- 內文與背景: 對比度 ≥ 4.5:1
- 使用 WebAIM Contrast Checker 驗證

### 字級
- 最小字級: 14px (0.875rem)
- 標題字級: ≥ 24px (1.5rem)
- 行高: ≥ 1.5

### 資訊傳達
- 不僅依賴顏色傳達資訊
- 使用圖示 + 文字
- 提供替代文字

## 印刷注意事項

### 色彩模式
- **印刷**: CMYK
- **數位**: RGB

### 出血
- 標準出血: 3mm
- 安全區域: 內縮 5mm

### 解析度
- 印刷品: 300 DPI
- 大型海報: 150 DPI
- 數位版: 72 DPI

## 檢查清單

印刷前檢查：
- [ ] 色彩模式正確 (CMYK)
- [ ] 解析度符合要求 (300 DPI)
- [ ] 出血設定正確 (3mm)
- [ ] 文字轉曲線
- [ ] 色彩對比度 ≥ 4.5:1
- [ ] 聯絡資訊正確
- [ ] QR Code 可掃描
- [ ] Logo 使用正確版本
