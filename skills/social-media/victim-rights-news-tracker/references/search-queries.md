## 搜尋引擎查詢語法

### Google 搜尋語法

```
# 精確短語搜尋（必須包含完整詞組）
"受害者家屬" "司法不公"

# 包含任一關鍵詞
受害者家屬 OR 被害者家屬 OR 被害人家屬

# 排除特定詞
"受害者家屬" -娛樂 -明星

# 限制網站
"受害者家屬" site:udn.com
"受害者家屬" site:ltn.com.tw
"受害者家屬" site:appledaily.com.tw

# 限制檔案類型（找尋 PDF 報告）
"被害人保護" filetype:pdf

# 時間限制（過去 24 小時）
"受害者家屬" after:2024-01-01

# 組合搜尋
"受害者家屬" (司法不公 OR 人權 OR 保護) site:cna.com.tw
```

### 新聞搜尋專用語法

```
# 中文新聞
site:news "受害者家屬"

# 找出相關人名
"受害者家屬" + "呼籲" + "表示"

# 判決相關新聞
"受害者家屬" + "判決" + (不服 OR 抗議 OR 遺憾)
```

## 搜尋策略建議

### 首次搜集策略

1. **廣度優先**：先使用較廣泛的關鍵詞（如「受害者家屬」）搜集大量結果
2. **深度挖掘**：對重要主題使用組合關鍵詞精確搜尋
3. **同義詞擴展**：交替使用「受害者/被害者/被害人」等同義詞
4. **時間分段**：長時間區段（如一年）建議分月搜尋，避免遺漏

### 每日更新策略

1. **時間限制**：使用「過去 24 小時」或「今日」篩選
2. **組合精簡**：每日搜尋使用較精確的組合，減少雜訊
3. **Tag 驅動**：優先搜尋已註冊的 Tag 相關新聞
4. **來源監控**：直接檢查主流媒體網站的最新新聞

## 查詢模板範例

### 模板 1：司法不公綜合搜尋

```
搜尋關鍵詞：
1. "受害者家屬" + "司法不公" + "台灣"
2. "被害者家屬" + "輕判" + "抗議"
3. "司法改革" + "被害人保護" + "新聞"
4. "victim family judicial injustice Taiwan"

預期結果：
- 家屬抗議輕判的報導
- 司法改革相關倡議
- 國際媒體對台灣司法的評論

Tag 建議：
#司法不公 #輕判爭議
```

```
- 相關記者會新聞

```

### 模板 3：特定案件追蹤搜尋

```
搜尋關鍵詞（以 [案件名稱] 為例）：
1. "[案件名稱]" + "家屬" + "最新"
2. "[案件名稱]" + "判決" + "上訴"
3. "[案件名稱]" + "賠償" + "和解"
4. 相關人名（被告、受害者、家屬）+ 新聞

時間限制：
- 首次：不限時間，搜集完整歷史
- 追蹤：過去 7 天或 30 天

預期結果：
- 案件最新進展
- 家屬相關發言或行動
- 司法程序更新

Tag 建議：
#[案件名稱] #案件追蹤
```

## 搜尋結果評估檢查清單

對每篇搜尋結果進行評估時，確認：

- [ ] **相關性**：是否與受害者家屬權益直接相關
- [ ] **地域性**：是否為台灣事件或與台灣相關
- [ ] **時效性**：發布日期是否在要求範圍內
- [ ] **完整性**：是否為完整報導（非僅標題或摘要）
- [ ] **來源可信度**：是否來自可信媒體或組織
- [ ] **內容品質**：是否有實質內容（非重複或抄襲）
- [ ] **獨特性**：是否為新資訊（非重複報導）

## 常見同義詞對照表

| 常用詞 | 同義詞 | 使用情境 |
|--------|--------|----------|
| 受害者 | 被害者、被害人 | 刑事司法語境常用「被害人」 |
| 家屬 | 家屬、家眷、親屬、親人 | 「家屬」最常用於新聞報導 |
| 權益 | 權利、人權、利益 | 「權益」較正式，「權利」較法律 |
| 司法不公 | 司法不公、司法偏頗、司法不正義 | 「司法不公」較常用 |
| 人權團體 | 民間團體、社運團體、NGO | 依組織性質選用 |

**使用提示**：執行搜尋時，建議同時使用 2-3 組關鍵詞，並根據搜尋結果調整策略。若某組關鍵詞結果過少，可改用較廣泛的同義詞。
