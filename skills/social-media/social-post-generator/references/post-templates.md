# 社群發文模板庫 (Post Templates Library)

本文檔提供各種風格的社群發文模板與範例，供 social-post-generator skill 使用。

## 一、嚴肅正式風格模板

### 模板 A1：修法進展報導

```
【司法改革的關鍵時刻】

[法案名稱] 修法進入重要階段，[具體進展描述]。

根據[機構/專家]統計/研究，[相關數據或事實陳述]。現行制度下，[具體制度缺陷說明]。

[權威人士/團體]表示：「[專家語錄]」

[改革訴求陳述]。請支持[具體行動呼籲]，讓我們一起為[目標願景]努力。

#司法改革 #[法案名稱] #制度正義 #被害人權益 #[組織名稱]
```

### 模板 A2：司法不公案件報導

```
【案件追蹤】[案件名稱] 再曝制度缺陷

[案件基本事實陳述，1-2句話]。然而，[司法不公具體表現]。

本案突顯了[制度層面問題]：
• [問題點1]
• [問題點2]
• [問題點3]

[專家/團體評論]：「[評論內容]」

[結論與呼籲]。正義不應該缺席。

#司法改革 #[案件名稱] #司法不公 #制度正義 #[組織名稱]
```

### 模板 A3：專家觀點分享

```
【法界觀點】[專家身分/職稱] 談 [議題]

[專家背景簡介]近日在[場合]發表對[議題]的看法：

```

## 六、範例組合

### 範例組合 1：修法新聞（三風格）

**素材**：刑事被害人保護法修正案排入議程

**A嚴肅正式**：
```
司法改革的關鍵時刻：刑事被害人保護法修正案排入本週議程

歷經民間團體多年的倡議，刑事被害人保護法修正案終於排入本週司法法制委員會議程。這是制度改革的關鍵一步。

根據司法院統計，2023年間僅有不到15%的重大刑事案件被害人獲得完善的訴訟協助...
```

**B情感訴求**：
```
「等了三年，終於等到這一天。」

這是小燈泡案家屬王婉諭在聽見修法排入議程時的第一句話。三年來，她從一位悲痛欲絕的母親，成為為所有被害人家屬發聲的倡議者...
```

**C行動呼籲**：
```
🚨 修法關鍵戰役！就在這週三 🚨

週三上午9點，刑事被害人保護法修正案將在司法法制委員會進行詢答！

但我們聽說反對勢力準備提出技術性延宕...

✅ 現在就做3件事：

1️⃣ 打電話給以下立委
2️⃣ 在委員會直播留言支持
3️⃣ 分享此貼文擴散

倒數2天！🔥
```

## 七、模板使用指南

### 選擇模板原則

| 新聞類型 | 推薦模板 | 理由 |
|---------|---------|------|
| 修法進展 | A1, C1 | 兼具嚴肅與行動 |
| 家屬訪談 | B1, B2 | 情感導向 |
| 判決結果 | D1, A2 | 需要專業分析 |
| 活動預告 | C2, C3 | 行動導向 |
| 國際案例 | D3, A3 | 需要對比分析 |
| 突破勝利 | D2, C | 慶祝+持續行動 |

### 模板組合技巧

1. **標題變化**：同一件新聞，三個標題角度不同
2. **重點選擇**：A選制度面，B選人情面，C選行動面
3. **圖片搭配**：A配新聞圖，B配家屬圖，C配海報圖
4. **CTA差異**：A邀請關注，B邀請共鳴，C邀請行動
