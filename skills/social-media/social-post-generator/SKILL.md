# 社群發文生成器 (Social Post Generator)

## 概述

此技能基於 **victim-rights-news-tracker**（受害者家屬權益新聞追蹤器）的每日更新報告，自動生成3個不同風格的社群媒體發文選項，供社群管理員選擇使用。

### 核心功能

1. **讀取新聞報告**：自動解析 victim-rights-news-tracker 產生的每日更新 Markdown 報告
2. **智能篩選**：從多則新聞中識別最重要、最具社群傳播價值的素材
3. **三風格生成**：
   - **選項A**：嚴肅正式風格（強調司法不公、權益保障）
   - **選項B**：情感訴求風格（強調家屬聲音、人文關懷）
   - **選項C**：行動呼籲風格（強調修法進展、民眾參與）
4. **素材配套**：每個選項包含完整發文文案、建議配圖、Hashtag、發布時間建議

### 適用平台

- Facebook（貼文、限時動態）
- Instagram（貼文、限時動態、Reels）
- Threads
- Line 官方帳號

---

## 觸發條件與使用時機

### 應立即使用此技能當：

1. **每日新聞追蹤完成後**
   - victim-rights-news-tracker 已產生當日更新報告
   - 範例：「請為今天的新聞報告生成社群發文選項」

2. **需要多風格選擇時**
   - 希望比較不同角度的發文效果
   - 範例：「生成3個不同風格的發文選項」

3. **特定主題推廣**
   - 針對特定案件或議題加強社群曝光
   - 範例：「針對#小燈泡案最新進展生成發文」

4. **多平台內容準備**
   - 需要同時準備多個平台的內容
   - 範例：「生成適合Facebook和Instagram的發文」

## 核心工作流程

### 工作流程 1：解析新聞報告 (News Parsing)

**使用時機**：讀取並解析 victim-rights-news-tracker 產生的報告

**執行步驟**：

1. **讀取報告檔案**
   - 預設路徑：`victim-news-daily-YYYYMMDD.md`
   - 或接受使用者提供檔案路徑

2. **識別重要新聞**
   
   從報告中篩選以下類別（按優先順序）：
   
   **高優先級**（必選）：
   - `[NEW]` 標記的新增報導
   - 立場變化追蹤（⭐星級變動）
   - 重大案件最新進展
   - 修法或政策突破
   
   **中優先級**（次選）：
   - 人權團體重要活動
   - 政黨/立委重要表態
   - 司法判決結果
   
   **低優先級**（補充）：
   - 國際相關案例
   - 背景分析報導

3. **擷取素材元素**
   
   對每則選中的新聞，擷取：
   - **標題**：簡短有力的標題
   - **核心事實**：3-5個關鍵資訊點
   - **情緒元素**：感人語錄、衝擊事實
   - **行動元素**：修法進度、參與方式
   - **圖片URL**：可供社群使用的圖片
   - **來源**：媒體名稱與原始連結
   - **關鍵人物**：涉及的角色（家屬、立委、法官等）

4. **建立素材清單**
   
   整理成結構化格式供後續使用。

### 重要：核心倡議知識引用

**當新聞內容涉及以下主題時，必須讀取 `references/victim-rights-9-demands.md`**：

- 被害人法庭地位、訴訟參與權
- 司法程序中的權利缺失（席位、上訴、詰問等）
- 刑事訴訟法修法相關新聞
- 家屬在法庭上的經歷與困境

**九大缺失與修法訴求的對應**：
- 九大缺失（席位、上訴、證據、詰問、抗告、光碟、問卷、迴避、評議簿）
- 全部對應 **「訴求1：強化被害人訴訟參與權」**
- 訴求1是為了解決這九大缺失而提出的總體修法方向

**引用原則**：
- 嚴肅正式風格（選項A）：可直接引用具體權利缺失名稱與法律條文，連結訴求1的修法方向
- 情感訴求風格（選項B）：將權利缺失轉化為「連...都沒有」的敘事，暗示訴求1的必要性
- 行動呼籲風格（選項C）：連結具體權利缺失與訴求1的修法倡議，呼籲支持修法

**避免**：
- 一次列出全部九項權利缺失（資訊過載）
- 未經消化直接複製法律條文
- 忽略「被害人不是當事人」這個根本原因
- 混淆「九大缺失」（程序問題）與「九大修法訴求」（立法方向）

### 工作流程 2：生成三風格發文 (Three-Style Generation)

**使用時機**：根據素材生成3個不同風格的發文選項

**執行步驟**：

#### 選項 A：嚴肅正式風格（Formal/Advocacy Style）

**定位**：專業、理性、訴求制度變革

**撰寫原則**：
- 突顯司法不公的具體事證
- 引用法律條文或專業術語
- 強調制度改革的必要性
- 引用權威人士（法官、律師、專家）觀點
- 維持客觀中立但堅定立場

**結構模板**：
```
【嚴肅正式】

[標題]：司法改革的關鍵時刻/【案件名稱】再曝制度缺陷

[開頭]：
事實陳述（1-2句）+ 制度缺陷點出（1句）

[正文]：
- 具體不公事證
- 修法進展現況
- 專家/團體呼籲

[結尾]：
改革訴求 + 行動呼籲

[Hashtag]：
#司法改革 #被害人權益 #制度正義 #[案件名稱]

[建議配圖]：
- 新聞截圖（嚴肅版面）
- 修法提案文檔
- 法院/立法院照片
- 數據圖表

[發布時機]：
週二至週四 10:00-12:00 或 19:00-21:00
```

**範例**：
```markdown
司法改革的關鍵時刻：刑事被害人保護法修法再次延宕

刑事被害人保護法修正案原定於上週進入司法法制委員會審查，卻因部分立委技術性阻擋而再次延宕。這已是該法案第3次無法順利進入審查程序。

根據民間司法改革基金會統計，目前僅有不到15%的重大刑事案件被害人能獲得完善的訴訟協助。現行制度下，被害人往往在司法程序中淪為「配角」，其權益與訴求難以獲得充分保障。

生命權平等協會理事長表示：「每一次延宕，都是對被害人家屬的再次傷害。我們呼籲司法法制委員會儘速排審此案，讓被害人權益保障不再只是空談。」

司法正義不應僅停留在口號。請支持#被害人權益保障修法，讓我們一起為制度正義發聲。

#司法改革 #被害人權益 #制度正義 #修法倡議
```

#### 選項 B：情感訴求風格（Emotional/Human-Interest Style）

**定位**：溫馨、感人、突顯人性與關懷

**撰寫原則**：
- 以家屬聲音為核心
- 使用第一人稱或引述語錄
- 突顯心理創傷與復原歷程
- 強調陪伴與支持的力量
- 引發共鳴與同理心

**結構模板**：
```
【情感訴求】

[標題]：「我只是想為孩子討一個公道」/ 當正義遲遲未到

[開頭]：
家屬語錄引用 + 情感場景描繪

[正文]：
- 家屬心路歷程
- 具體感人的小故事
- 支持系統的重要性
- 社會大眾的溫暖

[結尾]：
希望訊息 + 邀請關注陪伴

[Hashtag]：
#家屬的聲音 #正義不缺席 #陪伴 #司法溫暖 #[案件名稱]

[建議配圖]：
- 家屬溫馨照片（經授權）
- 紀念活動照片
- 手繪插圖/卡片
- 關懷活動紀錄

[發布時機]：
週五至週日 20:00-22:00（適合深度閱讀時段）
```

**範例**：
```markdown
「每天晚上，我都會夢到孩子問我：媽媽，為什麼壞人還沒有受到應有的懲罰？」

這是小燈泡案受害者母親王婉諭在最近一次訪談中的分享。作為一位失去孩子的母親，她不僅要面對椎心之痛，更要獨自在複雜的司法程序中艱難前行。

「很多人問我，為什麼要一直站出來？因為我知道，還有很多和我一樣的家屬，他們在黑暗中孤軍奮戰。如果可以，我希望成為他們的光。」

這些年來，我們見證了太多家屬在司法程序中的無奈與徬徨。他們需要的，不僅是正義的伸張，更是社會的理解與陪伴。

每一個案件背後，都是一個破碎的家庭。讓我們一起關注 #被害人家屬權益，用行動支持他們的勇氣與堅持。

因為正義不應該讓人等待太久。

#家屬的聲音 #正義不缺席 #司法溫暖 #小燈泡案 #陪伴是最長情的告白
```

#### 選項 C：行動呼籲風格（Call-to-Action/Mobilization Style）

**定位**：明確、直接、鼓勵參與行動

**撰寫原則**：
- 清晰的行動訴求（連署、參與活動、分享）
- 強調改變的可能性
- 提供具體參與方式
- 創造參與的緊迫感
- 鼓勵分享擴散

**結構模板**：
```
【行動呼籲】

[標題]：立刻行動！/ 你需要知道的3件事

[開頭]：
突破性進展/重要機會 + 時間緊迫性

[正文]：
- 具體行動項目（1-2-3步驟）
- 參與方式說明
- 過往行動成果
- 倒數計時/時間限制

[結尾]：
強烈呼籲 + 分享按鈕文字

[Hashtag]：
#一起行動 #支持修法 #改變從你開始 #[活動名稱] #立即分享

[建議配圖]：
- 活動海報
- 倒數計時圖片
- 行動步驟圖解
- 集氣/打氣視覺

[發布時機]：
週一至週三 12:00-13:00（午餐時間）或 17:00-18:00（下班前）
```

### 工作流程 3：素材整理與建議 (Asset Compilation)

**使用時機**：為3個發文選項整理配套素材

**執行步驟**：

1. **圖片素材分類**
   
   根據新聞報告中的圖片URL，分類為：
   
   | 類型 | 適用風格 | 說明 |
   |------|----------|------|
   | 新聞截圖 | A嚴肅 | 媒體報導畫面 |
   | 家屬照片 | B情感 | 溫馨、紀念活動 |
   | 活動現場 | C行動 | 記者會、抗議 |
   | 文檔/公文 | A嚴肅 | 修法提案、判決書 |
   | 人物特寫 | B情感 | 家屬訪談照片 |
   | 視覺圖表 | A嚴肅 | 數據統計圖 |
   | 呼籲海報 | C行動 | 活動宣傳圖 |
   
2. **Hashtag 組合建議**
   
   每個選項提供3組Hashtag（依字數限制選擇）：
   
   - **完整版**（適合Facebook）：8-12個標籤
   - **簡潔版**（適合Instagram）：5-8個標籤
   - **極簡版**（適合Threads）：3-5個標籤

3. **發布時間建議**
   
   根據平台特性與受眾行為：
   
   | 平台 | 最佳時段 | 說明 |
   |------|----------|------|
   | Facebook | 週二-四 10-12時、19-21時 | 工作日通勤與晚間 |
   | Instagram | 週五-日 20-22時 | 週末休閒時段 |
   | Threads | 每日 12-13時、21-22時 | 午餐與睡前 |

4. **互動預測與回應建議**
   
   預測可能的讀者反應，提供回應模板：
   - 正面支持回應
   - 質疑/反對意見回應
   - 詢問細節回應
   - 分享鼓勵回應

## 品質管理與最佳實踐

### 內容審核檢查清單

**發布前必須檢查**：

- [ ] **事實正確性**：所有數據、日期、人名已核實
- [ ] **家屬授權**：使用家屬照片或語錄已取得授權
- [ ] **中立平衡**：雖有立場但保持事實陳述客觀
- [ ] **法律安全**：無誹謗、無洩漏個資、無違反個資法
- [ ] **情緒適度**：情感訴求不過度消費悲劇
- [ ] **行動可行**：呼籲的行動項目具體且可執行

### 平台特性適配

**Facebook**：
- 字數限制：較寛鬆，可使用完整版文案
- 適合：深度討論、連結分享
- 互動：留言討論、分享擴散

**Instagram**：
- 字數限制：前2行最重要（折疊後）
- 適合：視覺導向、情感內容
- 互動：限時動態、Reels

**Threads**：
- 字數限制：500字，需更精簡
- 適合：即時性、輕鬆互動
- 互動：對話串、快速回應

### 發布策略建議

**單日單選項**：
- 一天僅發布1個選項（3選1）
- 其他選項保留備用或改日使用

**跨平台調整**：
- 同一內容可依平台調整字數與圖片比例
- 發布時間錯開（避免資訊轟炸）

**效果追蹤**：
- 記錄每個選項的互動數據（讚、留言、分享）
- 定期分析哪種風格最適合組織受眾
- 優化未來的生成策略

## 快速使用指南

### 基本使用

```bash
# 讀取今日新聞報告生成發文選項
python scripts/post_generator.py --input victim-news-daily-20240115.md

# 指定特定主題
python scripts/post_generator.py --input report.md --focus "#小燈泡案"

# 指定平台優先
python scripts/post_generator.py --input report.md --platform instagram
```

### 輸出檔案

產生的發文文件將保存為：
```
social-posts-YYYYMMDD.md
```

可直接交給社群管理員審核與發布。
