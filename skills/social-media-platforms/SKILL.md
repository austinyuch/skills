# 社群媒體平台優化器

將單一內容優化為 Facebook、Instagram、LINE 三平台的專屬版本。

## 快速開始

### 基本使用流程

1. **選擇平台** - 確認目標平台（Facebook / Instagram / LINE）
2. **讀取對應指南** - 參考 `references/{platform}.md` 了解平台特性
3. **套用優化規則** - 根據平台調整文案長度、hashtag、視覺規格
4. **產出最終內容** - 輸出符合平台特性的發文版本

### 平台選擇指引

| 需求 | 推薦平台 | 原因 |
|------|----------|------|
| 深度論述、長文分享 | Facebook | 支援 63,000 字，適合詳細報導 |
| 視覺導向、年輕族群 | Instagram | 圖片優先，18-35 歲主力 |
| 即時通知、高觸及率 | LINE | 訊息開啟率 80%+，適合公告 |

## 平台特性總覽

| 特性 | Facebook | Instagram | LINE |
|------|----------|-----------|------|
| **字數限制** | 63,000 | 2,200（前125字關鍵） | 5,000 |
| **主力用戶** | 30-55歲 | 18-35歲 | 全齡層 |
| **最佳字數** | 200-500 | 125-500 | 200-400 |
| **Hashtag** | 2-5 個 | 20-30 個 | 少用 |
| **發文時段** | 10-12, 19-21 | 20-22 | 18:00（週報） |
| **互動形式** | 分享、留言 | 讚、保存 | 點擊連結 |

## 注意事項

### 內容禁忌

| ❌ 禁止 | ✅ 替代方案 |
|---------|------------|
| 血腥、暴力畫面 | 使用象徵性圖示（法槌、天平） |
| 未經授權照片 | 使用剪影、背影、授權照片 |
| 過度政治攻擊 | 聚焦制度問題，非個人攻擊 |
| 煽動仇恨言論 | 理性訴求，訴說故事 |

### 法律合規

- 所有照片需經當事人書面授權
- 新聞畫面標註來源
- 人物面部模糊處理（如未同意）
- 避免誹謗言論
