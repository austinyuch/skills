# LINE 官方帳號申請與經營指南

LINE 在台灣擁有 2,100 萬活躍用戶，訊息開啟率 80%+，是 NPO 組織即時通知支持者的最佳管道。

## 平台特性

| 特性 | 說明 |
|------|------|
| **字數限制** | 5,000 字（實務建議 200-400 字） |
| **主力用戶** | 全齡層（18-50 歲滲透率 96%） |

## 官方帳號申請流程

### 準備文件

| 文件類型 | 說明 |
|----------|------|
| 法人登記證 | 內政部核發之社團法人登記證書 |
| 負責人身分證 | 理事長或執行長身分證正反面 |
| 銀行帳戶資訊 | 組織帳戶（用於未來金流） |
| 聯絡人資料 | 管理帳號的主要聯絡人 |

### 申請步驟

1. **註冊 LINE Business ID**
   - 瀏覽 [LINE Business ID](https://account.line.biz/login)
   - 使用 Email 註冊
   - 填寫組織名稱：生命權平等協會
   - 產業類別：非營利組織 / 社會服務

2. **申請官方帳號**
   - 點選「建立官方帳號」
   - **建議選擇「認證帳號（綠色盾牌）」**
     - 費用：免費申請，但需審核
     - 審核時間：5-10 個工作日
     - 優勢：增加可信度，解鎖進階功能

3. **填寫帳號資訊**

| 欄位 | 建議填寫內容 |
|------|-------------|
| 帳號名稱 | 生命權平等協會 |
| 帳號ID | @victim_rights_tw |
| 類別 | 非營利組織 |
| 描述 | 推動被害人權益保障，監督司法制度改革，為正義發聲 |
| 頭像 | 組織 LOGO（1080x1080px） |
| 封面圖 | 代表性活動照片或倡議主視覺 |

4. **提交認證申請**
   - 法人登記證書 PDF
   - 組織章程
   - 網站或社群媒體連結
   - 活動照片或新聞報導

## 常見問題

**Q: 免費版與付費版差異？**
A: 官方帳號免費，但「贊助訊息」推播需付費。一般推播（給已加好友者）免費。

**Q: 可以發送給非好友嗎？**
A: 需使用「贊助訊息」功能，按曝光或點擊付費。

**Q: 如何導流至網站？**
A: 在訊息中加入「動作按鈕」連結至網站，可追蹤點擊數據。

**Q: 認證被退件怎麼辦？**
A: 檢查文件是否齊全，組織網站/社群是否活躍，補充更多證明文件後重新申請。
