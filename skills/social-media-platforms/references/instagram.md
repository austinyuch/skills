# Instagram 視覺與發文指南

Instagram 是視覺導向平台，前 125 字決定用戶是否繼續閱讀。適合 18-35 歲族群，強調美感與情感連結。

## 平台特性

| 特性 | 說明 |
|------|------|
| **字數限制** | 2,200 字（前 125 字關鍵） |
| **主力用戶** | 18-35 歲 |
| **內容特性** | 視覺、美感、hashtag 分離 |
| **最佳時段** | 週五至週日 20:00-22:00 |
| **互動形式** | 讚、保存、分享 |
| **Hashtag** | 20-30 個（置底） |

## 文案結構範本

### 開場（前 125 字必抓眼球）

```
🚨 司法改革的關鍵時刻

監察院最新報告揭露：
僅12%被害人家屬能完整參與司法程序 😔
```

### 內文（簡短有力）

```
這代表什麼？
代表78%的家屬在法庭裡無法為親人發聲...

完整數據 👆 點擊連結
```

## 視覺識別系統

### 品牌色彩

```css
/* 主色調 */
--primary-blue: #1E3A5F;      /* 深藍：專業、信賴 */
--primary-gold: #C9A227;       /* 金色：珍貴的生命權 */

/* 輔助色 */
--accent-red: #B22222;          /* 警示紅：司法不公 */
--accent-white: #FFFFFF;      /* 純白：清白、正義 */
--warm-beige: #F5F5DC;        /* 溫暖米：家屬故事 */
```

### 色彩使用規則

| 用途 | 主色 | 輔助 | 背景 |
|------|------|------|------|
| 修法倡議 | 深藍 | 金色 | 白色 |
| 家屬故事 | 溫暖米 | 深藍 | 米色 |
| 行動呼籲 | 警示紅 | 白色 | 深藍 |

### 字體規範

- **標題**：Noto Sans TC Black（32-48px）
- **內文**：Noto Sans TC Regular（18-20px）
- **註解**：14-16px

## 圖片尺寸規格

| 類型 | 尺寸 | 比例 | 用途 |
|------|------|------|------|
| **方形** | 1080×1080px | 1:1 | 一般貼文 |
| **直向** | 1080×1350px | 4:5 | 故事型內容 |
| **輪播** | 1080×1920px | 9:16 | Stories、Reels |

### 檔案規格

- 格式：JPG（照片）、PNG（透明背景）、MP4（影片）
- 解析度：最低 1080px 寬度
- 檔案大小：JPG < 8MB，影片 < 100MB

## 內容類型範本

### Type A: 修法倡議（嚴肅正式）

**視覺特徵**：
- 深藍色背景或主視覺
- 金色作為強調色
- 簡潔有力的標語
- 數據圖表、時間軸

**範本設計**：
```
┌─────────────────────────┐
│                         │
│    [法槌/天平圖示]       │
│                         │
│   被害人權益保障法       │
│      三讀通過！          │
│                         │
│   ────────              │
│   這是歷史性的一刻       │
│                         │
│   #司法改革 #修法成功     │
│                         │
└─────────────────────────┘
```

### Type B: 家屬故事（情感訴求）

**視覺特徵**：
- 溫暖米色或柔和色調
- 照片為主，文字輔助
- 光暈、柔焦效果

**照片處理規範**：
- 統一使用暖色調濾鏡（色溫 +15，亮度 +10）
- 避免過度美顏，保持真實感
- 人物照片需經當事人授權
- 保護隱私：兒童面部可模糊處理

**禁止事項**：
- 不要使用黑白照片
- 不要過度濾鏡
- 不要未經授權使用照片

### Type C: 行動呼籲（高對比）

**視覺特徵**：
- 高對比配色（深藍+白，或紅+白）
- 箭頭、手指 icon
- 明確的 CTA
- 數字、倒數計時元素

## Reels 短影音規範

### 規格

- 尺寸：1080×1920px (9:16)
- 時長：15-90 秒（建議 30 秒內）
- 格式：MP4
- 字幕：必加（很多人靜音觀看）

### 內容類型

**1. 新聞快報型（15-30秒）**
- 場景：新聞畫面/資料畫面
- 配音：「今日重大消息，被害人權益保障法...」
- 字幕：同步顯示重點文字

**2. 家屬訴說型（60-90秒）**
- 場景：家屬訪談片段
- 風格：紀錄片風格，溫暖色調
- 配樂：溫柔鋼琴或弦樂

**3. 知識解說型（30-60秒）**
- 場景：圖文動畫 + 真人出鏡
- 內容：解釋被害人權益現況
- 風格：資訊圖表風格

## 素材管理

### 檔案命名規範

```
格式：YYYYMMDD_Type_Version.jpg

範例：
20260313_A_v1.jpg      ← 修法新聞正式版
20260313_B_v2.jpg      ← 家屬故事第二版
20260313_C_Reels_v1.mp4 ← 行動呼籲影片
```

## 禁忌與注意事項

| ❌ 禁止 | ✅ 替代方案 |
|---------|------------|
| 血腥、暴力畫面 | 使用象徵性圖示（法槌、天平） |
| 未經授權照片 | 使用剪影、背影、授權照片 |
| 過度政治攻擊 | 聚焦制度問題，非個人攻擊 |
| 煽動仇恨言論 | 理性訴求，訴說故事 |
| 過度商業化設計 | 維持 NPO 公益形象 |
