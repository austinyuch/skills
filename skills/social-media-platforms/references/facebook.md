# Facebook 發文優化指南

Facebook 適合深度內容、長文論述與社群討論。支援 63,000 字，是四平台中字數限制最寬鬆的。

## 平台特性

| 特性 | 說明 |
|------|------|
| **字數限制** | 63,000 字（實務建議 200-500 字） |
| **主力用戶** | 30-55 歲 |
| **內容特性** | 深度、社群、連結預覽 |
| **最佳時段** | 週二至週四 10:00-12:00, 19:00-21:00 |
| **互動形式** | 分享、留言討論 |
| **Hashtag** | 2-5 個 |

## 文案結構範本

### 開場（抓住注意）

```
🚨 司法改革的關鍵時刻

監察院最新調查：被害人家屬訴訟參與權長期被忽視
```

### 內文（詳細論述）

```
監察院12日發布最新調查報告，指出我國刑事訴訟程序中，被害人家屬的訴訟參與權長期遭到忽視。報告統計過去5年重大刑事案件中，僅12%的家屬獲得完整的程序參與機會。

根據相關統計與研究，現行制度下被害人往往在司法程序中面臨程序繁瑣與權益保障的不足。
```

### 數據呈現（視覺化）

```
📊 關鍵數據：
• 過去5年重大刑事案件：僅 12% 家屬獲得完整程序參與
• 78% 家屬表示無法充分表達意見
• 65% 認為判決結果未考量被害人權益
```

### 訴求/行動（明確 CTA）

```
💬 我們的訴求：
被害人及其家屬在司法程序中不僅要面對創傷，更要獨自面對制度的種種挑戰。我們呼籲：

1. 建立被害人訴訟參與制度
2. 強化檢察官與法院對被害人權益的保障
3. 提供家屬心理支持與法律協助

司法正義不應僅停留在口號。讓我們一起關注被害人權益保障，為制度正義發聲。
```

### 結尾（來源與連結）

```
📰 新聞來源：中央社
📅 報導日期：2026-03-12
🔗 完整報導：[點擊閱讀](https://www.cna.com.tw/news/...)
```

## 視覺素材規格

| 項目 | 規格 |
|------|------|
| **圖片尺寸** | 1200×630px |
| **影片長度** | 1-3 分鐘 |
| **設計風格** | 資訊圖表 |
| **檔案大小** | JPG < 8MB |

### 品牌色彩應用

- 主色：深藍 #1E3A5F（標題、重點）
- 輔色：金色 #C9A227（強調、數據）
- 背景：白色 #FFFFFF

## 發文策略

### 最佳時段

| 時段 | 適合內容 |
|------|----------|
| 10:00-12:00 | 修法進展、新聞報導 |
| 19:00-21:00 | 深度論述、家屬故事 |

### 互動設計

- **提問結尾**：「你怎麼看待這個議題？歡迎在下方留言 👇」
- **分享呼籲**：「請分享給更多人知道 🔄」
- **連結導流**：提供完整報導連結

## 成效指標

| KPI | 目標值 | 追蹤方式 |
|------|--------|----------|
| 分享數 | > 讚數 × 10% | Facebook Insights |
| 留言數 | > 讚數 × 5% | Insights |
| 連結點擊 | > 觸及數 × 3% | UTM 追蹤 |
